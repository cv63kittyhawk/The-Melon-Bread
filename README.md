🍈 The Melon Bread / めろんぱん

A tiny branching browser game about misunderstanding, innovation, and how people judge results after the fact.

Play as a planner or developer, make three decisions, and discover one of four endings.

⸻

✨ Features

* 🌍 Two UI worlds
    * 🇺🇸 Retro Future Edition (English)
    * 🇯🇵 ゆるUI版 (Japanese)
* 🎮 Branching story with 4 endings
* 📋 Share text copy feature
* 📱 Responsive single-file HTML
* 🎨 Theme switching + localized text

⸻

🎯 Concept

The same misunderstanding can be called:

* Creativity when it succeeds
* Failure when it doesn’t

This small game turns that idea into an interactive experience.

⸻

🚀 How to Run

1. Download or clone this repository
2. Open the_melon_bread_game.html in your browser

No build step. No dependencies. Just open and play.

⸻

🕹️ How to Play

1. Choose your world (English / Japanese)
2. Choose your role
3. Make 3 decisions
4. Reach one of 4 endings
5. Copy your share result

⸻

🧱 Tech Stack

* HTML
* CSS
* Vanilla JavaScript

Single file. Lightweight. Easy to remix.
